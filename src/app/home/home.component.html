Notes:

  • [queryParams] - one of the router directive where you can add query string parameter in the url
  • fragment - a directive that helps you to add # anchor in the query sting
  • routerLink - you can specify the url of the routed link that you place in appRoute in appModule.ts
  • [routerLink] - directive which you can add array of options for routed Links ['url/slug', ':id', 'edit/relativeSlugURL']
  • ActivatedRoute Object - it helps you to fetch data in query string or observ any changes in the routed link
